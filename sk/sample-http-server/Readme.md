# Exploring JavaScript on the Server

Init a simple Node project `pnpm init` and add some scripts:

```json
    "demo-one": "node http-server.mjs",
    "demo-two": "node non-blocking-http-server.mjs",
    "demo-three": "node routing-basic-http-server.mjs",
    "demo-four": "node routing-static-http-server.mjs"    
```
